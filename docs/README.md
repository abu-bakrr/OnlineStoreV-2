## 🚀 Быстрый старт

Выберите ваш вариант:

### Вариант 1: Автоматическая установка (3-5 минут) ⚡

```bash
ssh root@ВАШ_IP
GITHUB_REPO="https://github.com/USER/REPO.git" \
curl -fsSL https://raw.githubusercontent.com/USER/REPO/main/auto_deploy.sh | sudo bash
```

Подробнее: **[БЫСТРАЯ_УСТАНОВКА.md](БЫСТРАЯ_УСТАНОВКА.md)**

### Вариант 2: Интерактивная установка (5-7 минут)

```bash
ssh root@ВАШ_IP
wget https://raw.githubusercontent.com/USER/REPO/main/deploy_vps.sh
chmod +x deploy_vps.sh
sudo ./deploy_vps.sh
```

Подробнее: **[QUICK_START_RU.md](QUICK_START_RU.md)**

## 👨‍💼 Админ-панель

Доступна по адресу: `http://ваш-сайт/admin`

### Возможности:

- **Товары** — добавление, редактирование, удаление с загрузкой фото через Cloudinary
- **Категории** — полное управление категориями
- **Заказы** — просмотр, изменение статуса, чеки оплаты
- **Статистика** — пользователи, заказы, выручка, конверсия
- **Настройки** — Cloudinary, платежи, Telegram уведомления, SMTP
- **Менеджеры** — управление администраторами (только superadmin)

### Первый вход:

1. Зарегистрируйтесь на сайте
2. Перейдите на `/admin/login`
3. Если администраторов нет — вы станете первым админом

Подробнее: **[ПОЛНОЕ_РУКОВОДСТВО.md](ПОЛНОЕ_РУКОВОДСТВО.md#админ-панель)**

## 🎨 Кастомизация

Настройка магазина через файлы конфигурации:

1. **Основные настройки**: `../config/settings.json`
   - Название магазина
   - Логотип
   - Цвета
   - Валюта
   - Тексты интерфейса
   - Шрифты
   - Платежные системы

2. **Логотип**: `../config/logo.svg`
   - Замените на свой логотип

3. **Товары и категории**:
   - Через админ-панель `/admin`
   - Или напрямую в PostgreSQL

Подробнее в: **[ПОЛНОЕ_РУКОВОДСТВО.md](ПОЛНОЕ_РУКОВОДСТВО.md)** → раздел "Кастомизация"

## 🆘 Помощь

### Частые проблемы:

1. **Белая страница / 403** → `cd /home/shopapp/app && sudo ./fix_permissions.sh`
2. **Не запускается** → `journalctl -u shop-app -n 100`
3. **Git pull не работает** → `sudo -u shopapp git stash && sudo -u shopapp git pull`
4. **SSL не устанавливается** → Проверьте DNS: `ping ваш-домен.com`

Полный список решений: **[ПОЛНОЕ_РУКОВОДСТВО.md](ПОЛНОЕ_РУКОВОДСТВО.md#решение-проблем)**
